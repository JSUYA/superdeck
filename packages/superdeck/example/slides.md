# Present your widgets

SuperDeck allows you to present your widgets in a beautiful way.


```dart
  Example.simple(
    name: 'demo',
    builder: (args) {
      return Container(
        color: Colors.purple,
        child: const Center(
          child: Text('Hello World'),
        ),
      );
    },
  )
```
